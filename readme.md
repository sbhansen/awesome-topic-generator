# Passphrases are better than Pass Words
An app to generate pass phrases from a salt.

## Technology
1. React 16
1. Typescript
1. Webpack 4

## Development
### Install
1. clone code
1. install code<br>
`npm install`

### Scripts
- `npm run start`<br>
    Start the development server.
- `npm run dev`<br>
    Distribute development files.
- `npm run build`
    Distribute compressed files.
